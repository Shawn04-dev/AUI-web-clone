Notes on relative linking

  • If an HTML page is in Home/, link its CSS like: 
    <link rel="stylesheet" href="Home/style_hom.css">
  • From About/About.html link to its CSS as: 
    <link rel="stylesheet" href="About/style_abt.css">
  • From Department/Department.html link to its CSS as: 
    <link rel="stylesheet" href="Department/style_dep.css">
  • If you plan to serve the site from the repo root, the links above work as-is. If you move CSS into a shared folder (e.g., css/), update all pages to that new relative path.
